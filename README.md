ماڵپەڕی کوڕی پاک
ئەم ماڵپەڕە وەک ئەرشیفێکی تایبەت کاردەکات بۆ پاراستن و بڵاوکردنەوەی وتار و وانەکانی مامۆستا کوڕی پاک. لێرەدا دەتوانن بە ئاسانترین شێوە دەستتان بە سەرجەم ڤیدیۆکان بگات و بینەریان بن، بەو ئامانجەی قوتابییانی زانستی شەرعی و گشت موسڵمانان بتوانن سوودی لێ وەربگرن.
